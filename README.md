# 项目介绍 #
本项目是一个音频播放录音的开源代码，基于ELink407开发板。ELink407的主控是STM32F407VGT6，音频芯片使用的是LAN8720A，代码开发和编译环境为MDK5.10

src/Ethernet_ELink407 .....................源代码
